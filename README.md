# AutoClashBot

AutoClashBot is a Bot for Clash of Clans for automatization. The bot is using Bleustacks as a Virtual Device. 

### Features:
- [ ] Attack Dead Bases
- [ ] Train Army
- [ ] Activate Super Troops
- [ ] Donate Troops
- [ ] Play Clan Games
- [ ] Collect Resources
- [ ] Update Buildings
- [ ] Remove Obstacles
- [ ] Switch Accounts
- [ ] Switch Villages
- [ ] Attack in Night Village
